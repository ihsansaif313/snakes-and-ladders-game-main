# Snakes and Ladders Game

A classic Snakes and Ladders game built with HTML, CSS, and JavaScript. Play with friends, roll the dice, and race to the top while avoiding the snakes!

## Features

-   **Multiplayer Support:** Play with 2, 3, or 4 players.
-   **Custom Profiles:** Choose your player avatar and set your name.
-   **Interactive UI:** Smooth animations for dice rolls and player movement.
-   **Sound Effects:** Immersive audio for dice rolls, snake bites, ladder climbs, and winning.
-   **Responsive Design:** Playable on different screen sizes.

## How to Play

1.  **Select Players:** Choose the number of players (2, 3, or 4) on the home screen.
2.  **Setup Profiles:** Enter player names and select your avatars.
3.  **Start Game:** Click "Next" to enter the game board.
4.  **Roll Dice:** Click the dice icon next to your player card to roll.
5.  **Move:** Your piece will move automatically based on the dice roll.
    -   **Ladders:** If you land on the bottom of a ladder, you climb up!
    -   **Snakes:** If you land on the head of a snake, you slide down!
6.  **Win:** The first player to reach square 100 wins the game!

## Technologies Used

-   **HTML5:** For the game structure.
-   **CSS3:** For styling, animations, and responsive layout.
-   **JavaScript:** For game logic, state management, and DOM manipulation.

## Setup

1.  Clone the repository or download the source code.
2.  Open `index.html` in your web browser.
3.  Enjoy the game!
